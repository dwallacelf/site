**Networking, Observability, and Test Updates in the Calico/VPP v3.32.0 release on 5th of May 2026** -- The
Calico/VPP v3.32.0 release is now available, bringing updates across IPv6 networking, service handling, NAT,
policy testing, observability, and operational tooling. This release includes fixes and improvements for IPv6
neighbor handling, multicast forwarding, link-local address management, VXLAN routing, SRv6, CNAT, hostport
handling, BPF filtering support for capture commands, and support for VPP 26.02 (+ commits required pick up the
latest CNAT fixes).  It also includes new and expanded integration tests for Prometheus, services, policy, NAT,
ping, and single-stack IPv6 deployments.

Read the full release notes [here](https://github.com/projectcalico/vpp-dataplane/releases/tag/v3.32.0)!
